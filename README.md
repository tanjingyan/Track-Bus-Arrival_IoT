# Bus Arrival Monitor IoT

A Python-based IoT project that retrieves real-time bus arrival information from Singapore's LTA DataMall API and displays upcoming bus arrival timings. The project will later be integrated with a Raspberry Pi Pico W and LCD1602 display to create a physical smart bus arrival monitor.

## Features
- Real-time bus arrival information
- Displays next 3 arrival timings
- Handles non-operating bus services
- Auto-refresh bus data
- Planned Raspberry Pi Pico W and LCD1602 integration

## Technologies
- Python
- LTA DataMall API
- Requests
- Python Dotenv
- Git & GitHub
